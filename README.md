# react-mdui #

[MDUI][mdui] in ReactJS.

## Install ##

```sh
npm install react-mdui --save
```

## Usage ##

```sh
import { Button } from 'react-mdui';

const button = (
  <Button
    node="button"
    type="raised"
    block
    ripple
    disabled
  >
    Click Me!
  </Button>
);
```
